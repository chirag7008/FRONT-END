Basic HTML lists

Unordered list (`<ul>`)

Bulleted list — good when order doesn't matter.

  • Apples
  • Bananas
  • Grapes

Ordered list (`<ol>`)

Numbered list — good when order or steps matter.

  1. Preheat oven
  2. Mix ingredients
  3. Bake for 30 minutes

Start from 5 and use roman numerals:

  1. Step one
  2. Step two

Definition list (`<dl>`)

Pairs terms and definitions (good for glossaries).

HTML
HyperText Markup Language — the structure of web pages.
CSS
Cascading Style Sheets — styling for web pages.

Nested lists

You can put lists inside list items to create sub-lists.

  • Fruits
    • Apples
    • Oranges
  • Vegetables
    1. Carrot
    2. Spinach

Accessibility tip

Keep lists semantic (use proper <ul>, <ol>, <dl>) so screen readers and search engines understand your structure.
